# 运维AI助手 - 评测标准（Eval）

> 版本: v1.0  
> 更新日期: 2026-03-24

---

## 1. 评测目标

- 衡量诊断质量与稳定性
- 量化上线前后收益

---

## 2. 指标口径

| 指标 | 定义 |
|------|------|
| Top3命中率 | 最终根因是否落在Top3假设 |
| 建议可执行率 | 建议动作可直接执行占比 |
| 人工接管率 | 需人工全程接管事件占比 |
| 误导率 | 明显错误且影响处置的结论占比 |

---

## 3. 数据集

- 历史真实事件脱敏样本
- 分领域均衡：网络/数据库/应用/基础设施
- 按季度更新评测集版本

---

## 4. 通过标准

- Top3命中率 ≥ 75%
- 误导率 ≤ 5%
- 人工接管率 ≤ 35%
