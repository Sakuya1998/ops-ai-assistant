# 运维AI助手 - SRE运行与SLO

> 版本: v1.0  
> 更新日期: 2026-03-24

---

## 1. SLO定义

| 指标 | 目标 |
|------|------|
| 对话响应时延 P95 | < 5s |
| 告警初判时延 P95 | < 30s |
| 根因Top3命中率 | ≥ 75% |
| 高危误执行次数 | 0 |

---

## 2. 告警分级

| 级别 | 触发条件 | 响应要求 |
|------|---------|---------|
| P1 | 核心链路不可用 | 5分钟内接管 |
| P2 | 核心指标严重劣化 | 15分钟内缓解 |
| P3 | 非核心异常 | 班次内处理 |

---

## 3. 值班规则

- 值班人必须确认 incident 状态流转
- 高风险动作必须经 ops_admin 审批
- 处置完成后 24 小时内产出复盘
