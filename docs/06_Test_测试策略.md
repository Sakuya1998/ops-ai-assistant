# 运维AI助手 - 测试策略

> 版本: v1.0  
> 更新日期: 2026-03-24

---

## 1. 目标

- 验证 P0/P1 链路可用与正确
- 验证性能、稳定性和治理门禁

---

## 2. 测试分层

| 层级 | 目标 |
|------|------|
| 单元测试 | 节点逻辑与策略判断正确 |
| 集成测试 | API + 工具 + 存储链路可用 |
| 回放测试 | 历史故障样本下命中率达标 |
| 压力测试 | 高并发下时延与成功率达标 |

---

## 3. 关键用例

- `/chat` 返回结构化诊断
- `/alerts/webhook` 生成 incident
- 高风险动作未审批时被拒绝
- 动作失败触发回滚并写审计

---

## 4. 发布门槛

- P0用例通过率 = 100%
- 高风险误执行次数 = 0
- 根因Top3命中率 ≥ 75%
