# 运维AI助手 - 安全与合规

> 版本: v1.0  
> 更新日期: 2026-03-24

---

## 1. 安全目标

- 高危误执行为 0
- 所有执行动作全量审计可追溯
- 关键敏感信息不落日志明文

---

## 2. 权限模型

| 角色 | 权限 |
|------|------|
| ops_reader | 只读查询、查看报告 |
| ops_engineer | 执行中风险动作 |
| ops_admin | 审批并执行高风险动作 |

---

## 3. 控制策略

- 鉴权：OIDC/JWT
- 授权：RBAC + 最小权限
- 审批：高风险动作强制二次确认
- 脱敏：token/password/key 字段遮蔽
- 审计：记录操作者、时间、动作、结果

---

## 4. 合规留存

- 审计日志保留 3 年
- 关键事件不可篡改归档
- 月度安全审计抽检
