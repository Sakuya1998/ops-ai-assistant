# 运维AI助手 - Agent与Prompt策略

> 版本: v1.0  
> 更新日期: 2026-03-24

---

## 1. 角色划分

- SupervisorAgent：任务拆解与汇总
- NetworkAgent：网络与连通性
- DBAgent：数据库性能与连接
- AppAgent：应用异常与依赖
- InfraAgent：节点与资源调度

---

## 2. Prompt规范

- 强制证据化回答，不允许无依据结论
- 输出统一结构：结论、影响、证据、建议
- 低置信度场景必须提示不确定性
- 对高风险动作只给建议，不直接执行

---

## 3. 工具调用策略

- 先读后写：优先读取证据再建议动作
- 限制并发：单Agent同时最多3个外部调用
- 超时降级：依赖超时时返回部分可用结果
